# Employee Performance Analysis Dashboard

## Overview

Employee Performance Analysis Dashboard is a Data Analytics and Visualization project developed using Python, Streamlit, Pandas, NumPy, and Plotly.

The project analyzes employee performance data to identify trends in salary, attendance, experience, and departmental performance through interactive dashboards and visual analytics.

---

## Objectives

- Analyze employee performance metrics
- Identify salary and attendance patterns
- Compare departmental performance
- Generate data-driven insights
- Build an interactive analytics dashboard

---

## Features

### Dashboard Modules

- Home Dashboard
- Dataset Overview
- Data Cleaning
- Interactive Visualizations
- Key Insights
- Download Filtered Data

### Interactive Filters

- Department Filter
- City Filter
- Education Level Filter
- Performance Category Filter

### Analytics & Visualizations

- Histogram Analysis
- Box Plot Analysis
- Scatter Plots
- Line Charts
- Pie Charts
- Bubble Charts
- Correlation Heatmap
- Department Analysis
- City Analysis
- Attendance vs Performance Analysis
- Salary Trend Analysis

### Insights

- Top 10 Highest Paid Employees
- Top 10 Performers
- Average Salary Analysis
- Average Attendance Analysis
- Department-wise Performance Summary
